# DNA Analyzer

A simple bioinformatics project written in Python.

## Features

- DNA sequence input
- Nucleotide counting (A, T, G, C)
- Sequence length calculation
- GC content calculation

## Run

py dna_analyzer.py

## Example

Input:
ATGCGATCGATCGGCTA

Output:
Length: 17
A: 4
T: 4
G: 5
C: 4
GC Content: 52.94%
